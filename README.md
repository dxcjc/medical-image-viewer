# 医学影像处理系统

基于 Vue.js 的现代化医学影像渲染处理系统，支持 DICOM 格式的医学影像查看、处理和分析。

## 🏗️ 项目结构

```
medical-image/
├── medical-image-viewer/     # 前端应用
│   ├── src/                 # 源代码
│   ├── docs/               # 项目文档
│   ├── tests/              # 测试文件
│   └── package.json        # 依赖配置
├── docs/                   # 项目整体文档
├── scripts/                # 构建和部署脚本
└── README.md              # 项目说明
```

## 🚀 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm >= 8.0.0 或 yarn >= 1.22.0

### 安装依赖

```bash
cd medical-image-viewer
npm install
```

### 开发环境

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```
